======================
Asynchronous Requests
======================


Asynchronous Requests is an parallel requests tool for HTTP/Websocket protocols using Skitai_.


Installation
=============

**On Posix**

.. code-block:: bash

    apt-get install libxml2-dev libxslt-dev
    pip install aquests
    
**On Win32**

.. code-block:: bash

    pip install aquests


.. _Skitai: https://pypi.python.org/pypi/skitai
